# Hotel Finder

## Overview

Hotel Finder is a web application that helps users discover hotels based on their location, budget, and preferences. The platform provides an easy-to-use interface for searching and filtering hotels, making travel planning more convenient.

## Features

* Search hotels by location
* Budget-based hotel filtering
* Hotel rating filter
* Responsive design for mobile and desktop
* User-friendly interface
* Dark mode support
* Hotel booking form
* Google Maps integration
* User authentication (Login/Signup)
* Database integration for storing hotel information

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Node.js
* Express.js
* SQL Database
* Google Maps API
* Git & GitHub

## Installation

1. Clone the repository:
   git clone https://github.com/your-username/hotel-finder.git

2. Open the project folder.

3. Install dependencies:
   npm install

4. Start the server:
   npm start

5. Open the application in your browser.
